# CRM-Discovery
The CRM-Discovery is a microservice part of the CRM Service. It works as a Eureka server, allowing discovery to the other microservices.

## Built with

 * Spring framework
 * Eureka
 * Maven
 
### Related tools

 * Sonar
 * Jacoco
 * Pivotal Cloud Foundry
